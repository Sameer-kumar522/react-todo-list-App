# React Todo List App

A simple Todo List application built usingReact.js.  
This project helps users manage daily tasks with features like adding, editing, deleting, filtering, and tracking completion status.


## 🚀 Live Demo

https://sameer-kumar522.github.io/react-todo-list-App/

 # Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ❌ Delete tasks
- ✔️ Mark tasks as completed
- 🔄 Filter tasks (All / Completed / Pending)
- 📊 Task summary (Total / Completed / Pending)
- 💾 Data saved in browser using Local Storage


 # Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
